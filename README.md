# Scheduler
This project is to create a html-based schedule document. <br />
Helpful for people who work daily on Web Browser such as students studying remotely. <br />
<br />
<pre>
The format of the html document is: 
+----------------------------------+ 
|              [TITLE]             | 
|+-----------------------+ [LINKS] |
||      [SCHEDULE]       |         |
||                       |         |
||                       |         | 
||                       |         |
|+-----------------------+         | 
+----------------------------------+ 
</pre>
